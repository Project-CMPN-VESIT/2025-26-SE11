🚀 Siddhant SevaSetu – Smart Village Welfare Platform
Siddhant SevaSetu is a digital welfare platform developed in collaboration with Siddhant Samaj Vikas Sanstha, Ulhasnagar.
The project aims to improve rural welfare systems by digitizing elderly care, volunteer coordination, donation support, and education programs.

📌 Project Overview
In many rural areas, NGO activities are managed manually, leading to delays, poor tracking, and unorganized coordination.

Siddhant SevaSetu solves this by providing a centralized digital platform that:

Enables quick case reporting
Automatically identifies urgent cases
Connects volunteers and donors
Tracks welfare cases efficiently
Promotes education programs like MS-CIT and skill development

🎯 Key Features
🧑‍🤝‍🧑 Case Reporting
